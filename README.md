# agent-job-matcher

An AI agent (and companion frontend) that matches a given resume against a job posting, built with **Pydantic AI** (Python) for the agent logic and **Next.js** for the frontend. Observability is provided via **OpenObserve** and **Arize**. The project follows the **AI-DLC** (AI Development Lifecycle) methodology.

## Overview

- **Agent**: Python service built on [Pydantic AI](https://ai.pydantic.dev/) that evaluates resume-to-job-posting fit.
- **Frontend**: Next.js application for interacting with the agent.
- **Observability**: OpenObserve and Arize for tracing/monitoring agent behavior.
- **Methodology**: Developed following AI-DLC practices.
